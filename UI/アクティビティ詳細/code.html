週末の河川敷掃除とコーヒー

location_on 東京都台東区 隅田公園エリア

description イベント概要

休日の朝、少し早起きをして河川敷を散歩しながらゴミ拾いをしませんか？

「社会貢献」と意気込まず、あくまで散歩のついでに街をきれいにする、そんなゆるい活動です。作業の後は近くのカフェで美味しいコーヒーを飲みながら、少しだけ雑談をして解散します。

40代の参加者が多く、初参加の方もお一人様も大歓迎です。無理に話さなくても大丈夫、その場にいるだけでOKな雰囲気です。
